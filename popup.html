    请前往"风格"页面选择您喜欢的主题风格和颜色。
    支持秒、分钟、小时。Chrome 120+ 后台定时最短 30 秒，最长 24 小时。
    后台轮询由扩展后台单实例执行，不会因为打开多个页面而重复增加定时器；Chrome 120+ 下最短支持 30 秒。
    使用 OpenAI 兼容 Chat Completions 接口。仅在你点击“翻译邮件”时调用外部 API。API Key 只保存在本地扩展配置中，可随配置导入导出。
    打开邮件时会自动调用外部 API 提取更高置信度的验证码与验证链接。若选择复用翻译 API，下方独立提取 API 配置可留空；若 AI 提取失败，会保留本地规则结果，并可在邮件洞察区点击重试。
    秒
    默认 30 秒。到时会自动关闭姓名、密码、生日/年龄结果，也可以手动点击关闭。
